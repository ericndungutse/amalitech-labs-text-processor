# Amalitech Labs Text Processor

A JavaFX-based text processing tool developed for DataFlow Solutions.  
It supports regex-based **search**, **match**, and **replace** operations on plain text files with an intuitive desktop GUI.

## Features

- Open and edit text files
- Perform regex-based search and match
- Replace text using regular expressions
- View matches and highlights in real-time
- Efficient processing using Java Streams and Collections

## Technologies Used

- Java 17
- JavaFX
- Java Collections & Streams
- File I/O
- Regular Expressions (Regex)
